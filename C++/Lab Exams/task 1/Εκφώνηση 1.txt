ΆΣΚΗΣΗ 1
Α)
Να γίνει κλάση gate η οποία θα αναπαριστά μια λογική πύλη (π.χ. and, or, not κ.λ.π.) και
θα έχει ως ιδιωτικά μέλη μια περιγραφή (std::string) και τις ακόλουθες συναρτήσεις:
• κατασκευαστή κενό που θα θέτει στην περιγραφή το κενό string
• κατασκευαστή με όρισμα μια συμβολοσειρά που θα την θέτει ως αρχική τιμή
στην περιγραφή
• setter και getter για την περιγραφή (με ονόματα: setLabel, getLabel αντίστοιχα)
• συνάρτηση getOutput που θα επιστρέφει -1 (σημ. θα επικαλυφθεί στις
παράγωγες κλάσεις)
• συνάρτηση getPins που θα επιστρέφει 0 (σημ. θα επικαλυφθεί στις παράγωγες
κλάσεις)
• συνάρτηση setPin που θα δέχεται δύο ακέραια ορίσματα και δεν θα κάνει κάτι
(σημ. θα επικαλυφθεί στις παράγωγες κλάσεις)
• συνάρτηση getPin που θα δέχεται ένα ακέραιο όρισμα και θα επιστρέφει -1 (σημ.
θα επικαλυφθεί στις παράγωγες κλάσεις)
Β)
Να γίνει κλάση binaryGate που θα κληρονομεί την gate η οποία θα αναπαριστά μια λογική
πύλη με αποκλειστικά δύο εισόδους (π.χ. and, or). Η κλάση αυτή θα έχει ως ιδιωτικά μέλη
δύο ακέραιους αριθμούς που θα αντιστοιχούν στις τιμές εισόδου της πύλης, καθώς και τις
ακόλουθες συναρτήσεις:
• κατασκευαστή κενό που θα θέτει τις τιμές εισόδου στο -1
• κατασκευαστή με όρισμα μια συμβολοσειρά (την περιγραφή της πύλης), ο
οποίος θα θέτει τις τιμές εισόδου στο -1
• συνάρτηση getPins που θα επιστρέφει 2
• συνάρτηση setPin που θα δέχεται δύο ακέραια ορίσματα (αριθμό εισόδου και
τιμή) και θα θέτει στην αντίστοιχη είσοδο τη δοθείσα τιμή.
• συνάρτηση getPin που θα δέχεται ένα ακέραιο όρισμα (αριθμό εισόδου) και θα
επιστρέφει την τιμή της συγκεκριμένης εισόδου.
• (Σημείωση: Η πρώτη είσοδος αναφέρεται με το 0 και η δεύτερη με το 1).
Γ)
Να γίνει κλάση andGate που θα κληρονομεί την binaryGate η οποία θα αναπαριστά μια
λογική πύλη ΚΑΙ (AND) που θα περιλαμβάνει τις ακόλουθες συναρτήσεις:
• κατασκευαστή κενό
• κατασκευαστή με όρισμα μια συμβολοσειρά (την περιγραφή της πύλης)
• συνάρτηση getOutput η οποία θα επιστρέφει το αποτέλεσμα της πύλης στις τιμές
εισόδου της (Υπενθυμίζεται ότι η πύλη AND επιστρέφει 1 μόνο όταν και οι 2
είσοδοι έχουν τιμή 1, ενώ σε κάθε άλλη περίπτωση επιστρέφει 0).
Δ)
Να γίνει κλάση orGate που θα κληρονομεί την binaryGate η οποία θα αναπαριστά μια
λογική πύλη Ή (OR) που θα περιλαμβάνει τις ακόλουθες συναρτήσεις:
• Κατασκευαστή κενό
• Κατασκευαστή με όρισμα μια συμβολοσειρά (την περιγραφή της πύλης)
• συνάρτηση getOutput η οποία θα επιστρέφει το αποτέλεσμα της πύλης στις τιμές
εισόδου της (Υπενθυμίζεται ότι η πύλη OR επιστρέφει 1 όταν τουλάχιστον μία
από τις εισόδους της έχει τιμή 1, ενώ αν και οι δύο είσοδοι είναι 0 τότε
επιστρέφει 0).
