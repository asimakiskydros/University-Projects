ΆΣΚΗΣΗ 3
Να γίνει πρότυπη κλάση storage η οποία θα έχει ως ιδιωτικά μέλη ένα δυναμικό πίνακα
ενός γενικού τύπου δεδομένων και έναν ακέραιο που θα δείχνει το μέγεθος του
δυναμικού πίνακα. Η κλάση θα πρέπει να έχει έναν κατασκευαστή που θα δέχεται 2
ορίσματα (τη διεύθυνση ενός πίνακα του γενικού τύπου και το μέγεθος του) και θα πρέπει
να δεσμεύει δυναμικά χώρο στη μνήμη και να αντιγράφει τα δεδομένα του πίνακα που
δέχτηκε στα ορίσματα της (και το μέγεθος του). Επιπλέον, η συνάρτηση θα πρέπει να
υπερφορτώνει τον τελεστή [] για να επιστρέφει με αναφορά (για ανάγνωση και εγγραφή)
τα στοιχεία του πίνακα.
Έτσι για παράδειγμα η εντολή A[4] = τιμή1, θα θέτει την τιμή1 στη θέση 4 του πίνακα
και η cout<<A[3] θα εμφανίζει στην οθόνη την τιμή της θέσης 3 του πίνακα.
